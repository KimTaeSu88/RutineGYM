# 홈트레이닝 코어 빌더 앱 요구사항 정리 (Research)

## 1) 목표/가치 제안
- 사용자가 **오늘의 컨디션(난이도)**, **확보 가능한 시간**, **집중 타겟 부위**를 선택하면
  - 조합에 맞는 **맞춤형 코어 루틴**을 자동 생성/추천한다.
  - **층간소음 걱정 없이(조용한 동작 우선)** 매트 위에서 수행 가능한 옵션을 기본으로 제공한다.
- 루틴은 “**동작 N개를 이어서 1세트** → **총 세트 수 반복** → **세트 간 휴식**” 형태로 안내한다.

## 2) 입력(빌더 파라미터)
### 2.1 오늘 나의 체력 상태(난이도)
- **초급(🌱)**: 운동 막 시작/퇴근 후 피곤한 날  
  - 특징: 기본자세 위주, 세트 간 휴식 **60초**
- **중급(🌿)**: 땀을 조금 빼고 싶은 날  
  - 특징: 동작 속도 증가, 세트 간 휴식 **45초**
- **고급(🔥)**: 확실하게 불태우고 싶은 날  
  - 특징: 고강도 연속 동작, 세트 간 휴식 **30초**

### 2.2 확보 가능한 시간(세션 길이)
- **10분**: 짧고 굵게 끝내는 압축 코어 루틴(자기 전/샤워 직전)
- **20분**: 유산소 + 코어 혼합(표준)
- **30분**: 푸시업 보드 활용 포함 가능(상체 & 코어 전신 서킷)

> 참고: 예시에는 “15분” 케이스가 있어, 실제 제품에서는 **시간을 고정(10/20/30)**으로 할지  
> 또는 **5분 단위 커스텀(10~30)**을 허용할지 정책 결정이 필요.

### 2.3 집중 타겟 부위
- **전체 코어**: 플랭크, 마운틴 클라이머 (전반적인 뱃살 감소/체력 증진)
- **아랫뱃살**: 레그 레이즈, 플러터 킥 (아랫배 집중)
- **옆구리(러브핸들)**: 러시안 트위스트, 사이드 플랭크 (허리 라인)

## 3) 출력(루틴 생성 결과물 형태)
루틴 결과는 다음 요소를 포함한다.
- **루틴 이름**: 예) “초급 · 10분 · 아랫뱃살 집중”
- **구성 방식**:
  - (권장) **서킷형**: 동작 3~5개를 연속 수행 = 1세트
  - 세트 수: 시간/난이도에 따라 자동 산정(예: 3~4세트)
  - 세트 간 휴식: 난이도 규칙(초급 60s/중급 45s/고급 30s)
- **각 동작 상세**:
  - 동작명
  - 반복 수(회) 또는 시간(초)
  - 핵심 폼 포인트(짧게)
  - 주의사항(허리 뜸/목 긴장 등)
  - 참고 링크(유튜브/웹): “자세 영상/가이드 열기”
- **진행 타이머 옵션**:
  - 동작별 카운트다운(초 기반) + 세트 휴식 타이머
  - 회수 기반은 탭/음성/자동 카운트는 추후 확장

## 4) 루틴 생성 로직(핵심 규칙/가설)
### 4.1 난이도별 강도/휴식 규칙(명시)
- 초급: 폼 유지 중심, 휴식 60초
- 중급: 템포 증가/휴식 45초
- 고급: 고강도 연속/휴식 30초

### 4.2 시간별 세트/볼륨 규칙(초안)
- 10분:
  - 동작 3개 × 3세트(또는 4세트) 중심
  - 준비운동/정리운동은 최소(선택)
- 20분:
  - 코어 + 저충격 유산소(층간소음 고려 시 저충격 버전) 섞기
  - 동작 4개 × 3~4세트 등
- 30분:
  - 상체/코어/전신 서킷
  - 푸시업 보드(있을 경우) 동작을 1~2개 포함하는 “장비 옵션” 제공

### 4.3 타겟 부위 우선순위(초안)
- 전체 코어: 정적(플랭크류) + 동적(마운틴 클라이머류) 혼합
- 아랫뱃살: 하복부 자극(레그 레이즈/플러터 킥) 비중 증가 + 허리 보호 동작 보완
- 옆구리: 회전/측면 안정(러시안 트위스트/사이드 플랭크) 비중 증가

### 4.4 층간소음/조용한 모드(핵심 제약)
- 기본값: “**조용한 루틴 우선**”
- 고급/20~30분에서 유산소를 포함하더라도
  - 점프/버피 등 소음 큰 동작은 기본 제외하거나 “소음 허용” 토글에서만 노출
  - 대체 동작(저충격 마운틴 클라이머/스텝 변형 등) 제공

## 5) 운동 동작 카탈로그(초기 포함 후보)
> MVP는 “이름/타겟/난이도/소음/반복 타입(회/초)/주의사항” 정도만 메타데이터화해도 빌더가 동작 가능.

- 동작 메타데이터에 **자세 참고 링크(url)**를 포함(유튜브/웹 가이드).

- 코어/전체:
  - 플랭크(초), 마운틴 클라이머(회/초), 데드버그(회), 버드독(회)
- 아랫뱃살:
  - 레그 레이즈(회), 플러터 킥(초), 리버스 크런치(회)
- 옆구리:
  - 러시안 트위스트(회), 사이드 플랭크(초), 힙 딥(회)
- 상체(30분 옵션/푸시업 보드):
  - 푸시업(보드 그립 변형), 숄더 탭(회) 등

## 6) 화면/UX 요구사항(사용자 플로우)
### 6.1 빌더 입력 화면
- 3단계 선택 UI:
  - 난이도(초/중/고)
  - 시간(10/20/30분)
  - 타겟(전체/아랫뱃살/옆구리)
- 옵션(추가):
  - 조용한 모드(기본 ON)
  - 장비: 푸시업 보드 있음/없음
  - 통증/주의: 허리 민감/목 민감(동작 필터링에 사용)

### 6.2 결과 화면(루틴 카드)
- 세트 수, 세트 간 휴식, 동작 리스트를 한눈에
- “시작하기” 버튼 → 타이머/진행 화면
- 각 동작 항목에 “자세 영상/가이드 열기” 링크 버튼 제공
- “다시 생성” / “동작 교체” (동작 단위 스왑) 기능

### 6.3 진행 화면(타이머)
- 현재 동작, 남은 시간/남은 반복, 다음 동작 프리뷰
- 세트 완료/휴식 카운트다운
- 일시정지/종료

## 7) 예시 출력(요구사항의 기준 샘플)
**초급 / 15분 / 아랫뱃살 집중**(예시 규격)
- 1세트 = 아래 3개 동작 연속
  - 크런치 15회(허리는 바닥에 붙이고 상체는 살짝만)
  - 레그 레이즈 15회(허리 뜨지 않게)
  - 플랭크 30초
- 세트 간 휴식: 60초
- 총 3~4세트

> 제품화 시에는 “15분”이 허용되는지(커스텀) 또는 10/20/30으로 매핑할지 결정 필요.

## 8) 비기능 요구사항(NFR)
- **안전**: 허리/목 부담 큰 동작에는 경고와 대체 동작 제공
- **접근성**: 큰 글자 모드, 음성 안내(추후), 진동/소리 토글
- **오프라인**: 루틴 생성/타이머는 오프라인에서도 동작(추천)
- **로컬 저장**: 최근 루틴/즐겨찾기, 완료 기록(캘린더/스트릭은 추후)

## 9) MVP 범위 제안(1차 출시)
- 3단계 입력 → 루틴 생성 → 타이머 진행
- 동작 카탈로그(최소 12~18개) + 난이도/타겟/소음 태그
- 조용한 모드(기본 ON)와 푸시업 보드 옵션(30분에서만 영향)

## 10) 오픈 이슈/결정 필요
- 시간 선택 정책: 10/20/30 고정 vs 5분 단위 커스텀(10~30)
- “세트 수/동작 수/동작 시간(초)” 자동 산정 공식(난이도별 볼륨 정의)
- 유산소 포함 시 저충격 대체 동작 표준안
- 개인화 입력(초보/허리 통증/장비) 범위와 필터 규칙
